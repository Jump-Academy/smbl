# SourceMod Bot Library

Modular and extensible server-side bot library written in pure SourcePawn.

## Supported Games:
* *Team Fortress 2*

## Dependencies
* [Sourcemod 1.10](https://www.sourcemod.net/)
* [AutoExecConfig](https://github.com/Impact123/AutoExecConfig)
* [BotController](https://github.com/VoiDeD/sourcemod-botcontroller)
* [Multi Colors](https://github.com/Bara/Multi-Colors)
* [Octree](https://github.com/geominorai/sm-octree)
* [SMLib](https://github.com/bcserv/smlib/tree/transitional_syntax)
* [TF2Items](https://github.com/asherkin/TF2Items)
